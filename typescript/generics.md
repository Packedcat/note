## Generics

### 简介

- 使组件不仅能支持当前数据类型，也能支持外来的数据类型
- 使用`any`接收任何类型的参数回导致丢失部分类型信息
- 使用类型变量保存类型而不是值
- 可使用类型推论根据传入的参数自动判断泛型类型
- 创建泛型函数时，函数体内必须将这些参数当做任意或所有类型

### 使用泛型变量

创建参数为`T`类型数组的函数

```typescript
function loggingIdentity<T>(arg: T[]): T[] {
  console.log(arg.length);
  return arg;
}
```
